# QA Manual

## E2E Testing Flow
1. Install FocusLock.
2. Grant Permissions (Usage Access, Overlay, Family Controls).
3. Schedule a block for "YouTube" from 00:00 to 23:59.
4. Go to home screen and launch "YouTube".
5. Verify Native Shield / Overlay appears instantly.
6. Check Analytics dashboard to see +1 Block Attempt.
