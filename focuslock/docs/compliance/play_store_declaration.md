# Google Play Store Permissions Declaration

**Target Audience:** Google Play Review Team

**Permission 1:** `android.permission.PACKAGE_USAGE_STATS`
**Permission 2:** `android.permission.SYSTEM_ALERT_WINDOW`
**Permission 3:** `android.permission.FOREGROUND_SERVICE_SPECIAL_USE`

**Declaration Statement:**
"FocusLock is a Digital Wellbeing app. We require `PACKAGE_USAGE_STATS` to determine when a user opens an application they explicitly requested to block. We require `SYSTEM_ALERT_WINDOW` to display a blocking lock screen over the distracting application. We require `FOREGROUND_SERVICE_SPECIAL_USE` to maintain this monitoring while the user is actively engaged in a focus session. No data is sold or transmitted off-device to third parties."

**Required Action:** Provide a YouTube link to a screen recording demonstrating the user selecting an app to block, going to the launcher, and the overlay successfully blocking the app.
